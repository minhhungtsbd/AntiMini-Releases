# AntiMini-Releases

Kho lưu trữ phát hành chính thức cho **AntiMini Browser**.

Đây là kho lưu trữ công khai chứa các bản cài đặt đã được đóng gói và siêu dữ liệu (metadata) phục vụ cho cơ chế tự động cập nhật của ứng dụng AntiMini Browser. Kho lưu trữ chứa mã nguồn chính của phần mềm vẫn được giữ riêng tư (private).

## Các Nền Tảng Hỗ Trợ

* **Windows**: Bộ cài đặt định dạng `.exe` và `.msi`.
* **macOS**: Các gói ứng dụng `.dmg`.
* **Linux**: Các định dạng gói `.deb`, `.rpm`, và `.AppImage`.

## Luồng Tự Động Cập Nhật

Ứng dụng AntiMini Browser sẽ tự động gửi yêu cầu đến phần Releases công khai của kho lưu trữ này định kỳ để kiểm tra các phiên bản mới nhất và tự động tiến hành nâng cấp.

# AntiMini-Releases (English Version)

Official public release repository for **AntiMini Browser**.

This repository hosts the compiled installation packages and metadata for the AntiMini Browser auto-updater mechanism. The source code repository remains private.

## Supported Platforms

* **Windows**: `.exe` and `.msi` installers.
* **macOS**: `.dmg` bundles.
* **Linux**: `.deb`, `.rpm`, and `.AppImage` packages.

## Auto-Update Stream

The application automatically queries this repository's public releases to check for newer versions and perform updates.
